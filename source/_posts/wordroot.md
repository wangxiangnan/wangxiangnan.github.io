2. up：上面（方位）
2. de：下面（方位）
3. sub：一直...下去（精神）
4. in：在...内
5. ex：在...外
6. dis：否定
7. re：相反的、再一次
8. con/com：一起
9. per：每

### hum - 潮湿的 泥土
1. human：人类（西方认为人类是用泥土捏成的）
2. humid：潮湿的、湿热的
3. humble：adj-虚心的、卑微的 v-贬低、虚心
4. inhume: 埋葬
5. exhume：挖出
6. humiliate：羞辱、侮辱

### up - 向上

1. upbound adj. 上行的；上水的

up 上，向上 + bound 向…方向走 → 向上走 → 上行的

2. upbraid v. 谴责

up 上，向上 + braid 辫子 → 向上抓小辫子 → 谴责

3. upfrontad v./adj. 在前面（的）

up 上，向上 + front 前面 → 在前面（的）

4. upgrade v./n. 升级；上升

up 上，向上 + grad 等级 + e → 升级；上升

5. upheaval n. 动乱

up 上，向上 + heaval 举起 →〔矛盾〕上升 → 动乱

6. uphold v. 拥护

up 上，向上 + hold 支持 → 拥护

7. uproot v. 根除

up 上，向上 + root 根 → 根除

8. upturn n. （价格）上涨；好转

up 上，向上 + turn 转 → （价格）上涨；好转

### st stant stand 站着

2. understand: 理解
3. constant: adj-不变的 n-常量
4. distant: 遥远的、不同的
5. distance：距离
6. substance：物质基础

### sist stand 站着
1. assist v. 帮助

as 加强 + sist 站立 → 站起来帮助别人 → 帮助

2. assistant n. 助手

assist 帮助 + ant 表人 → 助手

3. consist v. 由…组成；与…一致

con 共同 + sist 站立 → 站到一起 → 由…组成

consistencen. 坚固性

4. con 共同 + sist 站立 + ence 表状态 → 坚定地站在一起 → 坚固性

5. desist v. 停止

de 加强 + sist 站立 → 站着不动 → 停止

6. exist v. 存在

ex 使… + ist〔= sist〕站立 → 使立足 → 存在

existencen. 存在

exist 存在 + ence 性质，状态，行为 → 存在

7. insist v. 坚持

in 使… + sist 站立 → 一直站着 → 坚持

insistencen. 坚持

insist 坚持 + ence 性质，状态，行为 → 坚持

irresistibleadj. 不可反抗的

ir 不 + resist 反抗 + ible 可…的 → 不可反抗的

8. persist v. 坚持到底

per 自始至终 + sist 站立 → 始终站着 → 坚持到底

persistentadj. 坚持的；顽固的

persist 坚持到底 + ent 具有…性质的，关于…的 → 坚持的；顽固的

9. resist v. 反抗

re 反 + sist 站立 → 站在对面 → 对立

resistancen. 反抗；抵抗力

resist 反抗 + ance 状态，性质，状况 → 反抗；抵抗力

10. subsist v. 生存

sub 下 + sist 站立 → 站下去 → 坚持活下去 → 生存

subsistencen. 生存

subsist 生存 + ence 表状态 → 生存

### cess 走

1. accede v. 答应，同意

ac 加强 + ced 行走，前进 + e → 一起走 → 同意

accessn. 接近；入口

ac 加强 + cess 行走，前进 → 一再向前走 → 接近

2. ancestor n. 祖先；祖宗

an 加强 + cest〔= cess〕行走，前进 + or 表人 → 走了很久的人 → 祖先

antecedentadj. 先行的；以前的

ante 前面 + ced 行走，前进 + ent 具有…性质的，关于…的 → 先行的；以前的

3. cede v. 割让；交出

ced 行走，前进 + e →〔使〕走开 → 割让

4. exceed v. 超越…的界限；胜过

ex 出 + ceed 行走，前进 → 走出去 → 超越

5. intercede v. 调停；仲裁

inter 在…之间 + ced 行走，前进 + e → 在两者中间走 → 调停

6. precede v. 在先，优于

pre 预先 + ced 行走，前进 + e → 在先，优于

precedentn. 先例；惯例

pre 前；预先 + ced 行走，前进 + ent 表名词 → 在前面的东西 → 先例

7. procedure n. 程序；手续

pro 在前 + ced 行走，前进 + ure 表结果 → 前面经过的〔历程〕→ 程序

proceedv. 继续进行，前进

pro 向前 + ceed 行走，前进 → 继续进行，前进

processn. 过程；进行

pro 向前 + cess 行走，前进 → 向前走的〔历程〕→ 过程

processionn. 队伍，行列

process 过程；进行 + ion 表名词 → 向前走 → 队伍

recedev. 后退

8. re 向后 + ced 行走，前进 + e → 后退

recessn. 休息；隐居

re 回 + cess 行走，前进 → 走回去 → 休息

recessionn. 撤退；交还

recess 休息；暂停 + ion 表动作 → 走回去 → 撤退

9. secede v. 脱离；退出

se 分开 + ced 行走，前进 + e → 分开走 → 脱离

10. succeed v. 成功；继续

suc 下 + ceed 行走，前进 → 从下面走到上面 → 成功

successn. 成功，胜利

suc 下 + cess 行走，前进 → 成功，胜利

11. unprecedented adj. 史无前例的

un 无 + precedent 前例 + ed 表形容词 → 史无前例的

### grad(i)= step, degree, to walk 步；度, 级；走 
1. centigradeadj. 百分度的

centi 百 + grad 步；度，级；走 + e → 百分度的

2. degradev. 降级；退化

de 向下 + grad 步；度，级；走 + e → 向下走 → 降级

3. downgradev. 贬低，降低

down 向下 + grad 步；度，级；走 + e → 向下走 → 降低

4. gradationn. 等级；层次

grad 步；度，级；走 + ation 行为，过程，状态，结果 → 等级；层次

5. graden. 等级；阶级

6. gradientadj. 步行的

gradi 步；度，级；走 + ent 具有…性质的，关于…的 → 步行的

7. gradientn. 坡度

8. gradualadj. 逐步的

grad 步；度，级；走 + ual 有…性质的 → 逐步的

9. graduate v./n. 毕业；分级

gradu〔= grad〕步；度，级；走 + ate 使… → 完成所有步骤 → 毕业

10. retrogradev. 退步；堕落

etro 向后 + grad 步；度，级；走 + e → 向后走 → 退步、

11. upgrade 升级  提高;改进
